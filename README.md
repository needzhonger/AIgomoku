# AIgomoku 项目 🎯 - 一款支持人机对战的 C++ 五子棋小游戏
欢迎来到 **AIgomoku** 项目！这是一个使用 `graphics.h` 图形库开发的五子棋小游戏，支持 **人机对战、双人对战、闯关模式** 三种玩法。项目专注于基础图形交互与 AI 落子策略的结合，适合算法学习与桌面游戏开发实践。

---

## ✨ 主要特性

### 🎮 三种对局模式

* **人机对战模式** 🤖
  玩家与 AI ，AI 采用自研的对抗搜索算法实现，具备一定博弈能力与局势判断。

* **双人对战模式** 👥
  支持本地双人交替落子（非联网），适合朋友间对战和教学使用。

* **闯关模式** 🧩
  提供一系列预设残局关卡，玩家需在指定条件下取胜，训练判断力与落子策略。

---

## 🧐 AI 引擎：对抗搜索策略

AIgomoku 的核心亮点是基于**对抗搜索**实现的简易 AI，包括以下关键设计：

* 局势评估：根据连子数量、活三、冲四等棋型打分
* 搜索算法：极大极小值搜索 + α-β剪枝（可设置搜索深度）
* 落子优先级排序，提升搜索效率与准确性
* 防守优先机制：主动消除潜在威胁，兼顾攻防平衡

## 🧩 项目结构概览（C++）

```
AIgomoku/
├— 源.cpp                // 程序入口，负责主循环与事件处理
├— GameManager.h / .cpp  // 五子棋核心逻辑（规则、胜负判定等）
├— AI.h / .cpp           // AI 实现模块（搜索、评估函数）
├— Player.h / .cpp       // 控制玩家和 AI 的行为
├— Chess.h / .cpp        // 控制棋盘的显示
└— res/                  // 图片、音乐、棋盘素材
```

## ⚙️ 编译与运行

### 环境要求

* Windows 系统（推荐）
* 支持 `graphics.h` 的 C++ 开发环境（如 Dev-C++、VC++ 等）
* 推荐使用 TDM-GCC 编译器，已配置图形库支持

### 运行方法
使用 Visual Studio 或使用你所在 IDE 运行 源.cpp 文件，确保支持 `graphics.h` 相关依赖。
